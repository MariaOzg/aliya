# Веб-платформа для медицинской клиники

Веб-приложение для медицинской клиники, обеспечивающее функциональность для пациентов, врачей и администраторов. Платформа объединяет фронтенд и бэкенд части в единое приложение.

## Основные функции

- Аутентификация и управление ролями (пациенты, врачи, администраторы)
- Просмотр медицинских услуг и их категорий
- Онлайн-запись на прием с выбором врача и времени
- Управление расписанием приемов
- Новости и акции клиники
- Образовательные материалы для пациентов
- Административная панель для управления контентом

## Технический стек

- **Фронтенд**: Next.js, React, TypeScript, Tailwind CSS
- **Бэкенд**: Next.js API Routes (Node.js)
- **База данных**: MongoDB
- **Аутентификация**: NextAuth.js

## Установка и запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/yourusername/aliya.git
cd aliya
```

2. Установить зависимости:

```bash
npm install
# или
yarn install
```

3. Настроить переменные окружения:
   - Создать файл `.env.local` в корне проекта
   - Добавить необходимые переменные:

```
MONGODB_URI=your_mongodb_uri
NEXTAUTH_SECRET=your_nextauth_secret
NEXTAUTH_URL=http://localhost:3000
```

4. Запустить проект в режиме разработки:

```bash
npm run dev
# или
yarn dev
```

5. Открыть [http://localhost:3000](http://localhost:3000) в браузере.

## Деплой

Проект настроен для деплоя на Render.com:

1. Создать аккаунт на [Render](https://render.com/)
2. Подключить GitHub-репозиторий
3. Настроить Web Service с следующими параметрами:
   - Build Command: `npm install && npm run build`
   - Start Command: `npm start`
   - Добавить переменные окружения

## Структура проекта

- `src/app` - Страницы приложения (App Router)
- `src/components` - React-компоненты
- `src/models` - Модели данных для MongoDB
- `src/lib` - Утилиты и вспомогательные функции
- `public` - Статические файлы
